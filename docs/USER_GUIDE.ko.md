# ScreenCapture 사용자 가이드

## 기본 화면

ScreenCapture는 하나의 메인 창에서 다음 세 탭을 제공합니다.

- **캡처**: 영역, 창, 전체 화면 캡처와 녹화
- **관리**: 저장된 이미지와 영상을 검색하고 미리보기
- **편집**: 이미지 주석, 영상 프레임 편집, 영상 압축

## 화면 캡처

캡처 탭에서 원하는 방식을 선택합니다.

- **영역 캡처**: 원하는 영역을 드래그해서 선택합니다.
- **창 캡처**: 열려 있는 창 목록에서 대상을 선택합니다.
- **전체 화면**: 마우스 커서가 놓인 모니터 전체를 캡처합니다.

영역 및 전체 화면 단축키를 누르면 키를 누른 순간의 마우스 커서 위치를 기준으로 대상 모니터가 결정됩니다. 모니터 선택 정보가 제공되지 않는 Windows 환경에서도 화면 순서와 source id를 이용해 올바른 모니터를 찾습니다.

## 화면 녹화

영역, 창, 전체 화면 녹화를 지원합니다. 영역 및 전체 화면 녹화도 시작 순간의 커서 위치 모니터를 기준으로 합니다. 녹화 중에는 Windows에서 재생되는 시스템 소리가 영상과 함께 녹음됩니다. 창 녹화도 선택한 창의 소리만 분리하는 방식이 아니라 PC 전체 출력 소리를 기록합니다. 마이크 입력은 현재 포함하지 않습니다. 녹화를 다시 실행하거나 정지 버튼을 누르면 녹화가 종료되고 설정에 따라 MP4/GIF로 변환됩니다.

## 관리 탭

- 파일을 선택하면 오른쪽에서 이미지 또는 영상을 미리 볼 수 있습니다.
- **이름 변경**은 앱 내부 대화상자에서 처리되며 확장자를 포함한 파일명을 확인한 뒤 적용할 수 있습니다.
- `↑`/`↓`: 이전 또는 다음 파일 선택
- `Home`/`End`: 목록의 첫 번째 또는 마지막 파일 선택
- 선택 항목이 화면 밖으로 이동하면 목록이 자동으로 스크롤됩니다.
- 마우스 오른쪽 버튼 메뉴에서 파일 탐색기의 해당 파일 위치를 열 수 있습니다.
- 이미지와 영상은 각각 이미지 편집기 또는 프레임 편집기로 전달할 수 있습니다.

## 편집 탭

### 이미지 편집

이미지를 불러온 뒤 펜, 화살표, 사각형, 원, 텍스트, 모자이크 등의 주석 도구를 사용할 수 있습니다. 저장 시 파일 탐색기에서 위치와 파일명을 직접 선택합니다.

- 도형은 드래그 시작점과 현재 커서를 두 모서리로 사용하므로 Windows 화면 배율과 관계없이 커서 위치에 맞게 생성됩니다.
- 도형 도구는 여러 주석을 연속해서 그릴 수 있도록 유지됩니다. 기존 도형을 클릭하거나 드래그하면 선택 도구로 자동 전환됩니다.
- 선택한 도형은 이동하거나 모서리 핸들로 크기를 조절할 수 있습니다.
- `Ctrl+Z` 또는 상단의 **실행 취소** 버튼으로 도형·펜·텍스트·모자이크 추가, 이동, 크기 조절, 색상 변경, 주석 삭제를 되돌릴 수 있습니다.
- 최대 50개의 이미지 편집 상태를 실행 취소 기록으로 유지합니다. 새 이미지를 열면 이전 이미지의 기록은 초기화됩니다.

### 프레임 편집

영상을 프레임 목록으로 열어 불필요한 프레임을 삭제하고 GIF 또는 MP4로 내보낼 수 있습니다. 방향키와 `Home`, `End`, `Page Up`, `Page Down`으로 프레임을 이동할 수 있습니다.

- `Ctrl+Z`를 누르면 가장 최근에 삭제한 프레임과 당시 선택 위치가 복원됩니다.
- 새 영상을 열면 이전 영상의 실행 취소 기록은 초기화됩니다.

### 영상 압축

영상을 선택하고 압축 설정을 적용해 별도 파일로 저장합니다.

## 전역 단축키

기본 단축키는 다음과 같습니다.

| 동작 | 기본 단축키 |
| --- | --- |
| 영역 캡처 | `Ctrl+Shift+1` |
| 창 캡처 | `Ctrl+Shift+2` |
| 전체 화면 캡처 | `Ctrl+Shift+3` |
| 영역 녹화 | `Ctrl+Shift+4` |
| 창 녹화 | `Ctrl+Shift+5` |
| 전체 화면 녹화 | `Ctrl+Shift+R` |

설정에서 **변경**을 누른 뒤 보조키를 포함한 전체 조합을 입력할 수 있습니다. 단축키 입력을 취소하거나 다른 프로그램으로 전환하면 기존 전역 단축키가 자동으로 복구됩니다.

## 트레이 상주

상단의 **트레이 상주** 토글을 켜면 메인 창을 닫아도 프로그램과 전역 단축키가 계속 실행됩니다. Windows가 트레이 아이콘을 숨겨진 아이콘 영역에 배치할 수 있습니다. 토글을 끈 상태에서 창을 닫으면 프로그램이 완전히 종료되며 전역 단축키도 동작하지 않습니다.

## 저장 위치

기본 저장 폴더는 `사진\ScreenCapture`입니다. 설정에서 저장 위치와 이미지 형식, 녹화 FPS 및 내보내기 형식을 변경할 수 있습니다.
