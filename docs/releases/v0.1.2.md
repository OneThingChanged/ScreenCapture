# ScreenCapture v0.1.2

This patch improves keyboard navigation in the media manager and restores global shortcuts reliably after shortcut editing.

## Changes

- Added Arrow Up and Arrow Down navigation to the Manage file list.
- Added Home and End navigation for jumping to the first or last visible file.
- Automatically scrolls the selected media row into view and updates the preview.
- Keeps normal arrow-key behavior inside search fields and video controls.
- Restores global shortcuts when shortcut input loses focus or the app becomes inactive.
- Reports failed or invalid global shortcut registrations in the application log.

## Downloads

- Use `ScreenCapture-Setup-0.1.2-x64.exe` for installation and automatic updates.
- Use `ScreenCapture-Portable-0.1.2-x64.exe` to run ScreenCapture without installation.

Windows SmartScreen may display a warning because this release is not Authenticode-signed.
