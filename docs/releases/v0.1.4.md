# ScreenCapture v0.1.4

This release improves high-DPI image editing, annotation interaction, undo support, and media management.

## Changes

- Preserves physical image pixels when screenshots and imported images are transferred into the editor.
- Displays the complete source image at the correct aspect ratio instead of rendering only a scaled upper-left section.
- Aligns rectangle, ellipse, and mosaic geometry with the pointer at non-100% Windows display scales.
- Uses top-left shape origins and stroke-aware dimensions so drag endpoints match the visible annotation bounds.
- Keeps drawing tools active for repeated annotations and automatically enters selection behavior when an existing object is targeted.
- Adds `Ctrl+Z` undo history for image annotations, movement, resizing, color changes, mosaic operations, and annotation clearing.
- Adds `Ctrl+Z` restoration for deleted video frames, including the previous selection position.
- Replaces the native filename prompt with an in-app rename dialog that reports validation and filesystem errors.
- Adds an editor rendering self-test for high-DPI regression checks.

## Downloads

- Use `ScreenCapture-Setup-0.1.4-x64.exe` for installation and automatic updates.
- Use `ScreenCapture-Portable-0.1.4-x64.exe` to run ScreenCapture without installation.

Windows SmartScreen may display a warning because this release is not Authenticode-signed.
