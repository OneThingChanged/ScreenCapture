# ScreenCapture v0.1.5

This release adds Windows system-audio recording to every screen recording mode.

## Changes

- Captures Windows system output audio with Electron's loopback display-media stream.
- Includes system audio in region, window, and full-screen recordings.
- Preserves audio tracks when region video is cropped through a canvas stream.
- Records WebM audio with Opus and converts MP4 audio to AAC at 192 kbps.
- Reports an explicit recording error instead of silently producing a video without an audio track when Windows loopback audio is unavailable.
- Extends recording self-tests to cover both full-screen and cropped-region audio output.

## Downloads

- Use `ScreenCapture-Setup-0.1.5-x64.exe` for installation and automatic updates.
- Use `ScreenCapture-Portable-0.1.5-x64.exe` to run ScreenCapture without installation.

Windows SmartScreen may display a warning because this release is not Authenticode-signed.
