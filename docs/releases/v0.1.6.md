# ScreenCapture v0.1.6

This release records both Windows system output and the default microphone in every screen recording mode.

## Changes

- Captures Windows system output through Electron's loopback display-media stream.
- Captures the Windows default microphone alongside system audio.
- Mixes both inputs into one level-managed audio track with Web Audio before recording.
- Includes the mixed audio in region, window, and full-screen recordings.
- Preserves mixed audio when region video is cropped through a canvas stream.
- Records WebM audio with Opus and converts MP4 audio to AAC at 192 kbps.
- Uses a dedicated recorder session for scoped media permissions.
- Reports an explicit error instead of silently producing a video without either required audio input.

## Downloads

- Use `ScreenCapture-Setup-0.1.6-x64.exe` for installation and automatic updates.
- Use `ScreenCapture-Portable-0.1.6-x64.exe` to run ScreenCapture without installation.

Windows SmartScreen may display a warning because this release is not Authenticode-signed.
