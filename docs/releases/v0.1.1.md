# ScreenCapture v0.1.1

ScreenCapture now uses a persistent dark-themed workspace that keeps capture, media management, and editing in one application window.

## Highlights

- Added a tab-based Capture, Manage, and Edit workspace.
- Added region, window, and full-screen recording alongside the three screenshot modes.
- Kept the main window available while capture and recording selectors run separately.
- Added six configurable global shortcuts with full modifier-key support.
- Added media previews, in-app video playback, scrolling, search, and Reveal in File Explorer.
- Added image import and Save As support to the image editor.
- Added keyboard navigation and automatic list scrolling to the frame editor.
- Added Save As dialogs for GIF and MP4 exports.
- Added an optional tray toggle: closing the window either keeps ScreenCapture in the tray or exits immediately.
- Improved media URL handling and recording reliability.
- Added English and Korean README documentation.

## Downloads

- Use `ScreenCapture-Setup-0.1.1-x64.exe` for installation and automatic updates.
- Use `ScreenCapture-Portable-0.1.1-x64.exe` to run ScreenCapture without installation.

Windows SmartScreen may display a warning because this release is not Authenticode-signed.
