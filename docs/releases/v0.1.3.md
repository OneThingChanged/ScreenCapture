# ScreenCapture v0.1.3

This release improves multi-monitor capture targeting, tray reliability, and documentation.

## Changes

- Uses the monitor under the mouse cursor when a region or full-screen shortcut is triggered.
- Captures the cursor monitor at shortcut invocation time so later pointer movement cannot change the target.
- Adds robust Windows screen-source matching when `display_id` is unavailable.
- Applies the same cursor-monitor behavior to region and full-screen recording.
- Re-registers the Windows tray icon when the main window closes to tray.
- Adds high-DPI tray image representations and a clear `ScreenCapture` tooltip.
- Removes the redundant capture-after-action controls from the visible UI while preserving stored-setting compatibility.
- Adds organized user, architecture, and release documentation under `docs`.

## Downloads

- Use `ScreenCapture-Setup-0.1.3-x64.exe` for installation and automatic updates.
- Use `ScreenCapture-Portable-0.1.3-x64.exe` to run ScreenCapture without installation.

Windows SmartScreen may display a warning because this release is not Authenticode-signed.
